***Техническое задание***

**1. Цель разработки**

Разработать расширение (плагин) для Visual Studio Code, предназначенное для автоматической обработки комментариев в выделенном фрагменте текста. Плагин должен облегчать добавление, удаление и форматирование комментариев, обеспечивая единый стиль оформления в коде.

**2. Функциональные требования**

Расширение должно поддерживать следующие операции:

2.1. Добавление комментариев
- Добавление символа # перед каждой строкой выделения.
- Пустая строка заменяется на строку, содержащую только #.

2.2. Удаление комментариев
- Удаление символа #, а также пробела после него (если присутствует).
- Сохранение исходных отступов (ведущих пробелов).

2.3. Форматирование комментариев
- Перенос строк комментария по ширине 80 символов.
- Сохранение исходных отступов и структуры текста.
- Форматирование применяется только к строкам, начинающимся с #.

2.4. Работа с многострочным выделением
Плагин должен корректно обрабатывать выделения, содержащие одну или несколько строк.

2.5. Поддержка горячих клавиш
- Ctrl + Alt + C — добавление комментариев
- Ctrl + Alt + X — удаление комментариев
- Ctrl + Alt + F — форматирование комментариев

**3. Ограничения**
- Плагин работает только с символом # как маркером комментариев.
- Форматирование переносит строки только по словам (word wrap).
- Функции применяются только к выделенному тексту.

**4. Среда разработки**
- Visual Studio Code версии
- Язык разработки: JavaScript
