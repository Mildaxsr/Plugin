# Comment Formatter — расширение для Visual Studio Code

---

##  Описание

**Comment Formatter** — это расширение для Visual Studio Code, предназначенное для автоматической обработки комментариев в выделенном фрагменте текста.

Плагин позволяет:

- добавлять комментарии (`#`) к выбранным строкам;
- удалять существующие комментарии;
- форматировать длинные комментарии, перенося их по ширине 80 символов;
- работать с многострочными выделениями.

Расширение упрощает работу с кодом и помогает автоматически поддерживать единый стиль комментариев.

---

##  Возможности расширения

### 1. Добавление комментариев

- Добавляет `#` перед каждой строкой выделения.
- Пустые строки заменяются на `#`.
- Уже закомментированные строки не изменяются.

**Команда:**  
`commentFormatter.addComments`

---

### 2. Удаление комментариев

- Удаляет ведущий `#` (и пробел после него).
- Сохраняет исходные отступы.
- Строки без комментария остаются без изменений.

**Команда:**  
`commentFormatter.removeComments`

---

### 3. Форматирование комментариев

- Переносит комментарии по словам до ширины 80 символов.
- Сохраняет отступы.
- Форматирует только строки, начинающиеся с `#`.

**Команда:**  
`commentFormatter.formatComments`

---

##  Горячие клавиши

| Действие                | Горячая клавиша |
|------------------------|-----------------|
| Добавить комментарии   | Ctrl + Alt + C  |
| Удалить комментарии    | Ctrl + Alt + X  |
| Форматировать          | Ctrl + Alt + F  |

*(Хоткеи можно изменить в настройках VS Code.)*
