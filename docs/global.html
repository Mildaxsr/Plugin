Функция активации расширения. Вызывается VS Code при первом запуске любой из команд расширения. Здесь регистрируются три команды: - commentFormatter.addComments - commentFormatter.removeComments - commentFormatter.formatComments

addComments(lines) → {Array.<string>}

Добавляет символ "# " перед каждой строкой. Пустая строка превращается в строку, содержащую только "#". Уже закомментированные строки (начинающиеся с "#", игнорируя отступы) остаются без изменений.

formatComments(lines, widthopt) → {Array.<string>}

Форматирует комментарии так, чтобы длина строки не превышала заданную ширину. Обрабатываются только строки, начинающиеся с "#". Отступы в начале строки сохраняются. Текст комментария переносится по словам (word wrap).
